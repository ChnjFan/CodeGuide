# STL 基础

STL（Standard Template Library）标准模板库是C++标准库的核心部分，提供一套通用的数据结构和算法。

## STL 设计理念：泛型编程

STL 采用泛型编程的思想将数据结构和算法彻底分离，使用同一套算法操作不同的数据结构。

- 面向对象编程：强调数据和操作封装在一起。
- 泛型编程：数据和算法分离，通过统一接口连接。

泛型编程更适合处理通用的数据结构和算法问题，OOP 更适合处理领域建模和业务逻辑。

STL 通过模板（template）实现泛型，可以处理不同类型的数据；通过迭代器（iterator）实现解耦，让算法可以操作任意容器，不需要知道容器的内部实现。

## 核心组件

STL 主要由六个组件组成，其中容器、迭代器和算法是最核心的三大组件。

- 容器（Container）：负责存储数据的类模板。
