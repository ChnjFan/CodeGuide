# 回溯

回溯算法（Backtracking）是**暴力搜索算法的一种**，也叫 “试探法”，核心思想是：**不满足条件就回头，换一条路再试**，本质是通过**递归**遍历所有可能的解空间，找到所有符合要求的解。

 ## 解决问题

回溯算法解决找到所有可能的组合、排列和路径问题：

1. 需要一步一步选择
2. 选择后进入新的子问题
3. 选择错误后返回上一步正确的状态，尝试其他选择

适用场景：

- 组合问题：数组中选择 k 个数输出所有组合。
- 排列问题：数组元素全排列问题。
- 子集问题：数组所有的子集。
- 棋盘问题：N 皇后，解数独。
- 路径问题：迷宫、矩阵中的路径。

## 核心逻辑

1. 路径：已经做出的选择
2. 选择列表：可以选择的元素
3. 结束条件，路径满足条件，保存结果

```cpp
vector<int> result;
void dfs(vector<vector<int>> &res, vector<int>& path) {
    if (istarget(path)) { // 路径满足条件添加到结果中
        res.push_back(path);
        return;
    }
    // 遍历所有可能
    for (auto& element : elements) {
        path.push_back(element);// 做出选择加入路径中
        dfs(res, path);// 递归判断是否满足
        path.pop_back();// 回退到上一个状态，尝试下一个选择
    }
}
```
