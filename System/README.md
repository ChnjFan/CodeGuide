## 操作系统概念

操作系统（OS）管理计算机硬件与软件资源的核心系统软件，负责调度资源、提供交互接口，保证所有组件协调高效运行。

### 进程和线程管理

创建、调度、终止进程 / 线程，分配 CPU 执行时间，实现多任务并发。

- [MiniGDB：掌握 GDB 核心原理](./Proc/minigdb.md)：GDB 调试工具允许在程序运行期间暂停程序、检查变量状态、逐行执行代码和修改内存和寄存器。

### 内存管理

分配、回收内存空间，避免内存泄漏，支持虚拟内存。

### I/O 与文件管理

操作系统的 I/O 管理负责协调计算机的 **输入 / 输出设备**（如磁盘、键盘、网卡、打印机、显示器等）与 CPU、内存之间的数据传输。

- [C10K 问题：异步 IO 与多路复用](./IO/multiplexing.md)：高效的 I/O 处理机制，允许单个进程（或线程）同时监视多个 I/O 文件描述符，并在其中任何一个描述符就绪时进行处理。

文件管理定义文件的存储格式、访问权限（如只读、可写），组织目录结构，方便用户和程序查找、管理文件。
